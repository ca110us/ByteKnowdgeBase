# Linux HowTos

以下内容基于 Ubuntu Server。

# 进程

# 如何批量杀死含某个关键词的进程？

ps -ef | grep PS_KEYWORDS | grep -v grep | cut -c 9-15 | xargs kill -9

# 用户帐号篇

# 如何查看用户所在的用户组？

groups USERNAME

# 如何创建新用户帐号？

useradd USERNAME -m -s /bin/bash

参数：

  • -m: 建立home目录
  • -s: 指定shell

# 如何设置用户帐号密码？

passwd USERNAME

# 如何赋予新用户 sudo 权限？

将用户加入sudo用户组即可。

sudo usermod -a -G sudo USERNAME

参数：

  • -a: Append模式
  • -G: 指定要增加的group

# 如何生成 SSH Key ？

ssh-keygen -t rsa -C "user@machine"

生成的 SSH KEY 在目录~/.ssh/下。

# 如何复制本机的 SSH 公钥到远程主机？

使得本机可以免密码登陆远程主机。

ssh-copy-id [-i [identity_file]] [user@]machine

# 时区

# 如何设置服务器时区？

sudo timedatectl set-timezone Asia/Shanghai

参考：

How To Set Up Timezone and NTP Synchronization on Ubuntu 14.04

# 如何修改网卡名称？

网卡我们习惯上会用eth0, eth1...ethN的方式来命名，但最新的ubuntu server对网卡的命名会出来一些奇怪的名字，比如enp0s3。如果你看着不爽，就可以给他改名。改名的步骤如下：

  1. 编辑/etc/default/grub，修改：
GRUB_CMDLINE_LINUX=""

为

GRUB_CMDLINE_LINUX="net.ifnames=0 biosdevname=0"

  1. sudo update-grub

  2. 编辑，/etc/network/interfaces，你会看到如下类似内容：

auto lo
iface lo inet loopback

auto enp0s3 
iface enp0s3 inet dhcp

将奇怪的网卡名enp0s3修改为eth0。如果还有第二块网卡那么，第二块网卡为eth1。

# 如何将网卡设置为静态IP ?

编辑/etc/network/interfaces，下例把eth1网卡IP设置为了192.169.59.11。

iface eth1 inet static
address 192.168.59.11
gateway 192.168.59.1
netmask 255.255.255.0

重启：sudo reboot
